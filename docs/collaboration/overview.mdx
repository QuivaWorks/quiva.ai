# Collaboration

QuivaWorks enables seamless team collaboration on AI-powered tasks. Share assistant sessions, collaborate in real-time, and stay informed with intelligent notifications.

## Key Features

- **Shared Sessions** – Invite team members to collaborate on assistant conversations
- **@Mentions** – Use @mention syntax to add team members to sessions
- **Real-Time Updates** – See live updates as teammates work on shared sessions
- **Smart Notifications** – Get alerts when team members complete tasks or reply
- **Unread Tracking** – Never miss important updates with automatic badge counts
- **Shared With Me** – Find all sessions shared with you in one organized place

## Getting Started

- **New to collaboration?** Start with [Collaborative Sessions](/collaborative-sessions)
