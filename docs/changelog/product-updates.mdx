# QuivaWorks v1.0.5 Release Notes
<Note>
**Release Date:** February 6, 2026  
**Version:** v1.0.5  
**Release Type:** Patch Release  
</Note>
## Overview
This patch release focuses on enhancing your Assistant experience with improved session persistence, expanded marketplace capabilities, better mobile support, and more reliable integrations. We've streamlined the assistant marketplace with new project management and research tools, improved your ability to maintain conversations across sessions, and made critical improvements to authentication and integration management.
## ‚ú® New Features & Enhancements
### Assistant Session Persistence ‚≠ê **Major Enhancement**
Your Assistant conversations are now automatically saved and can be restored anytime:
- ‚úÖ **Resume Conversations** - Close the app and come back later to continue exactly where you left off
- ‚úÖ **Multiple Chat Tabs** - Keep multiple conversations with the same Assistant open simultaneously and switch between them
- ‚úÖ **Recent Sessions** - Quick access to your recently used chats without scrolling through history
- ‚úÖ **Automatic Prompt Saving** - Your input is automatically saved, so you won't lose what you were typing
- ‚úÖ **Works Across Devices** - Your sessions sync to the backend, so you can access them from any device
- ‚úÖ **Mobile-Optimized** - Responsive design ensures session management works smoothly on phones and tablets
### New Assistants in the Marketplace ü§ñ
We've expanded the marketplace with powerful new assistants to help you manage projects, track research, and stay informed:
- ‚úÖ **GitHub Release Manager** - Automate your release process with semantic versioning, commit analysis, and professional release notes generation
- ‚úÖ **GitHub Issues Manager** - Streamline issue tracking, prioritization, and workflow management directly from QuivaWorks
- ‚úÖ **Incident Report Manager** - Document, track, and manage incident reports with structured workflows and team collaboration
- ‚úÖ **Company News Tracker** - Research companies of interest with automated news aggregation for portfolio tracking, reputation management, and competitive intelligence
**Easier Downloads:** The experience of downloading assistants into your account has been improved, making them ready to use immediately without additional configuration.
### Mobile View Enhancements üì±
The mobile experience has been significantly improved with better navigation, responsive layouts, and iOS-specific fixes:
- ‚úÖ **Collapsible Sidebar** - The sidebar now intelligently collapses on mobile devices, giving you more screen space for your conversations. Tap the hamburger menu to reveal it
- ‚úÖ **Improved Touch Interaction** - Fixed font sizing and keyboard behavior to prevent unwanted zoom on iOS devices, making typing smoother and more natural
- ‚úÖ **Responsive Tab Management** - Chat tabs now display as a dropdown menu on mobile instead of a horizontal list, saving precious screen space
- ‚úÖ **Better Tool Output Display** - Large tool outputs (like JSON data or tables) are now collapsible with "Show More/Less" buttons, keeping conversations clean and readable
- ‚úÖ **Optimized Prompt Input** - The message input area now expands properly on all mobile devices, giving you more room to type longer messages
- ‚úÖ **Cleaner Navigation** - The main navigation now clearly labels "Chat" and includes a "New Chat" button for quick access to starting fresh conversations
- ‚úÖ **Dynamic Viewport Handling** - Fixed iOS-specific issues with viewport height and body positioning to ensure the interface adapts properly when the keyboard appears
- ‚úÖ **Icon-Only Headers** - Text labels are hidden on mobile to maximize space, with icons providing clear visual cues for each action
### Vision Tool Improvements
The vision tool now includes intelligent content type validation to ensure you're using the right tool for the right file type:
- ‚úÖ **Better Error Messages** - If you accidentally try to analyze a PDF, Word document, or spreadsheet with the vision tool, you'll now get a helpful message suggesting you use the knowledge search tool instead
- ‚úÖ **Faster Analysis** - The system now validates file types before processing, preventing wasted time on unsupported formats
- ‚úÖ **Memory Optimization** - Fixed a memory leak in the vision cache that could cause performance degradation over time
### Integration & OAuth Improvements üîó
Integrations are now more reliable and easier to manage with improved authentication configuration and connection management:
#### OAuth Connection Management
- ‚úÖ **Edit Redirect URLs** - You can now modify the redirect URL field when configuring OAuth integrations, giving you more flexibility in setting up third-party connections
- ‚úÖ **Delete Integrations Reliably** - Fixed critical bugs that were preventing users from successfully deleting OAuth connections. You can now cleanly remove integrations from your account
- ‚úÖ **Delete Confirmation Warning** - When deleting an integration connection that multiple Assistants are using, you'll now see a warning: "If other assistants are using this connection, they will no longer be able to use it and will need to be updated." This prevents accidental disruption to your workflows
#### Better Integration Discovery
- ‚úÖ **Visible Auth URLs** - Integration authentication URLs (auth_url and token_url) are now displayed in your account settings when available, making it easier to understand what's required for setup
- ‚úÖ **Cleaner Connection Cards** - Integration connections now display with an "Active" indicator badge and icon-based controls, making it immediately clear which integrations are connected and working
#### Improved Integration Stability
- ‚úÖ **Flexible Authentication Types** - The system now supports a wider range of authentication methods (API Key, Basic Auth, Bearer Token, OAuth) with better detection and handling
- ‚úÖ **Better Error Handling** - Fixed underlying issues with connection deletion and authentication configuration that were causing integration failures
- ‚úÖ **Personal vs. Shared Connections** - Clear distinction between personal integrations (only you can use) and shared integrations (your entire team can use)
## ‚ö†Ô∏è Breaking Changes
### Claude Models Only
We've streamlined our LLM provider support to focus on Claude models, which provide the best performance and reliability for QuivaWorks:
- **OpenAI and Gemini models have been removed** - If you're currently using OpenAI or Gemini models with your assistants, you'll need to reconfigure them to use Claude models
- **Action Required:** Update any assistants that were using OpenAI or Gemini to use Claude (Claude 3.5 Sonnet is recommended for best performance)
- **Why:** This change allows us to optimize the platform specifically for Claude's capabilities and provide better performance, reliability, and cost efficiency
### Deprecated Gemini Embedding Model
- **Gemini embeddings have been removed** - If you're using Gemini for knowledge base embeddings, please migrate to an alternative embedding provider before upgrading
- **Supported alternatives:** Claude embeddings or other configured providers

- ‚úÖ **Credit Usage Tracking** - Fixed accuracy of credit usage reporting to ensure billing reflects actual usage
- ‚úÖ **Knowledge File Naming** - Resolved conflicts with knowledge file naming that could cause retrieval issues
- ‚úÖ **Cross-Account Search** - Improved search functionality when working with multiple accounts
- ‚úÖ **Secret Key Creation** - Fixed issues preventing proper creation of secret keys for API authentication
- ‚úÖ **API Key Validation** - Enhanced validation of API keys to prevent invalid configurations
- ‚úÖ **Indexer Improvements** - Fixed underlying indexing issues that could affect knowledge retrieval performance
- ‚úÖ **Assistant Download** - Fixed bugs preventing successful download of assistants from the marketplace
