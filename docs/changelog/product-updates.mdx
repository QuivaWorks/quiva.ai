We're excited to announce **QuivaWorks v1.0.4**, a focused patch release designed to enhance your experience across the platform. This release includes 35+ improvements spanning knowledge management, agent capabilities, navigation, and marketplace features.

## üéØ Key Highlights
### Enhanced Knowledge Management
Improved knowledge upload and management workflows make it easier to organize and utilize your documentation. Better integration with Mintlify documentation systems ensures seamless content handling.
### Smarter Agent Features
Experience improved agent interfaces with enhanced streaming capabilities, better model selection tools, and improved agent-to-agent communication for more sophisticated automations.
### Redesigned Navigation
New sidebar navigation system provides a cleaner, more intuitive way to explore QuivaWorks. Navigate faster and find what you need with our improved UX.
### Marketplace Expansion
Deploy marketplace agents directly to your new accounts with improved integration between the marketplace and your core platform.
